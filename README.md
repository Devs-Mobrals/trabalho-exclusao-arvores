## 📚 Sobre o Projeto

Este projeto foi desenvolvido em Java com o objetivo de implementar e demonstrar:

- Inserção em Árvore B
- Exclusão em Árvore B
- Inserção em Árvore B+
- Exclusão em Árvore B+
- Inserção em Árvore Rubro-Negra
- Exclusão em Árvore Rubro-Negra
- Balanceamento automático
- Rotações
- Redistribuição
- Merge de nós
- Correção de propriedades da árvore
- Range Query
- Encadeamento entre folhas

O projeto possui implementação completa das estruturas e impressão visual da árvore no console.

---

# 🌳 Estruturas Implementadas

## Árvore B

A implementação da Árvore B contém:

- Inserção
- Split de nós
- Remoção em folhas
- Remoção em nós internos
- Redistribuição pela esquerda
- Redistribuição pela direita
- Merge
- Busca de predecessor
- Busca de sucessor
- Impressão por níveis

### Classes

- `ArvoreB`
- `NoB`
- `Main`

---

## 🌿 Árvore B+

A implementação da Árvore B+ contém:

- Inserção
- Exclusão
- Split
- Merge
- Redistribuição
- Encadeamento entre folhas
- Range Query
- Distinção entre chaves internas e folhas
- Impressão das folhas encadeadas
- Ponteiros `next`

### Classes

- `ArvoreBPlus`
- `NoBPlus`
- `Main`

---

## 🔴⚫ Árvore Rubro-Negra

A implementação da Árvore Rubro-Negra contém:

- Inserção
- Correção pós-inserção
- Rotação à esquerda
- Rotação à direita
- Remoção
- Correção pós-remoção
- Casos de balanceamento
- Nó NIL sentinela
- Impressão visual da árvore

### Classes

- `ArvoreRubroNegra`
- `No`
- `Main`
